智能晾衣架仿真

功能：1）采集温度、湿度、雨滴占比、光照强度等传感器数据
               可通过反复按下按键“切换界面”切换查看当前数据及晾衣架收缩条件
          2）可通过按键“设置建”切换需要更改的数据条件，并通过“加”“减”键改变此数据数值
          3）可通过“手动控制”更换操作模式，当页面“Auto”变为“RC”时，代表切换为手动模式
               并可通过“加”“减”键控制电机正转（CCW）反转（CW），最大圈数5
          4）当任意传感器数值超过限制值时，并且电机圈数不等于0（已伸出），会进入强制收回状态
               操作模式变“Auto”，如果雨滴百分比造成收回，会在收回时进行报警，收回后报警结束
